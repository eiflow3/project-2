The main goal of this Offline GUI Application is to move on with the manual tracking of daily orders using pen and log book.


This GUI app when initialized into a personal laptop will be having a setup phase where it will asked all your products(name, prices, additional columns) to set up the database. Then it will ask the user setting up to be the master account of this offline application asking for username and password saving the credentials into the database. 

Starting up the application:
GUI will asked for credentials;
Set as admin on default (since this is the first time setup);
Gather username;
Ask if want passoword or pin;

Then setting up the products;
One by one the user will add products with it's name|unit-cost|selling-price|quantity(skippable set default to 1)

This app should make a business track orders going out of their store easily (let's start by the user typing the basic information of the customer e.g. address | name | product alias | quantity | price | delivery rider (optional) | statsus ).
